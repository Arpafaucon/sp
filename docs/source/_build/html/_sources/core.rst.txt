SP - Core package
=================

Description
-----------

The **sp_core** package contains the common materials for the SPCF stack.

* maps
* Rviz setup
* shell scripts
* python utilities modules
* Common messages

``sp_core.tools`` python module
-------------------------------

``sp_core.tools.rospools`` submodule
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

This python modules provides Service and Publisher pools
to deal with similar requests regarding different namespaces.
